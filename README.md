[www.kingchiller.com](http://www.kingchiller.com)

Website for the *Kingchiller* festival in Basel.

Development
-----------
Install dependencies:
```
bundle install
npm install
bower install
```

Build website and serve at [localhost:4000](http://localhost:4000):
```
grunt develop
```
